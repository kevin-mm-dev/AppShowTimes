# showtimes
Aplicacion con crud a un servidor de Node, donde puedes guardar peliculas que tengan un titulo, una descripcion y una imagen.
It's not so fun but it was fun create it.

## Start Project
```
npm install
```
then
```
npm run serve
```

### Start server
```
cd server
```
then
```
npm run dev
```


### Used technology
- Vue
- Node
- TS
- Bootstrap
